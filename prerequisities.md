# EzeeCanteen Application - Prerequisites

## Prerequisites

### System Requirements
- **Operating System**: Windows 10 or newer
- **Storage**: 500MB free disk space
- **Display**: 1366x768 resolution minimum

### Network Requirements
- Active internet connection for database connectivity
- Stable local network for printer and device communication
- All devices must be connected to the same network
- Firewall exceptions for the application and it's required ports

### Database Requirements
- Database driver installed and up-to-date 

### Hardware
- Compatible receipt printers
